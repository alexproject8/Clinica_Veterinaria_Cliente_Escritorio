Prologo

Esta aplicación consta de varias funcionalidades, de entre las cuales destacan el alta para clientes, el alta de pacientes/mascotas, consultas de mascotas, clientes y citas, se puede eliminar mascotas, dar de baja clientes, modifcarlos, anular y modificar citas, consultar historial clínico, crearlo y generar informe de historial, además se pueden generar facturas.
